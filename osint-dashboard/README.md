# OSINT Threat Intelligence Dashboard

## Overview
This project is a full-stack OSINT intelligence platform designed to simulate real-world reconnaissance workflows used in cybersecurity operations.

## Features
- Subdomain enumeration via crt.sh
- WHOIS data extraction
- Data persistence using SQLite
- Interactive dashboard interface
- Modular architecture for extensibility

## Technologies
- Python (Flask)
- SQLite
- HTML/CSS
- Chart.js

## Use Case
Security analysts use this tool to quickly aggregate public information about a target domain to identify attack surfaces.

## Example Workflow
1. Input target domain
2. Enumerate subdomains
3. Extract ownership data
4. Visualize results

## Future Improvements
- Shodan API integration
- Email exposure detection
- Risk scoring engine
